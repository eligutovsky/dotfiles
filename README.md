# Dotfiles

## [Homebrew](https://brew.sh/index_ru.html)

### Instalation

```sh
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" 
$ brew bundle
```

### Tips

Search an app
```sh
$ brew search <query>
```

## [Oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)

Copy file

```sh
$ cp .zshrc $HOME/.zshrc
```
